## 3. 데이터 소개
 
 - 슈퍼마켓의 매출과 매장의 크기, 일일 이용자 수의 경계르 분석한 데이터이다.
    
    - 상점 ID: (색인) 특정 상점의 ID

    - Store_Area: 야드 광장에 있는 상점의 물리적 영역입니다.

    - Items_Available: 해당 상점에서 사용할 수 있는 다른 항목의 수

    - 일별 고객 수: 한 달 동안 평균 매장을 방문한 고객 수

    - Store_Sales: 매장에서 발생한 (US $) 매출
